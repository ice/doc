Class Ice

Dump

Dumps information about a variable(s) 

  $foo = 123;
  echo (new \Ice\Dump())->variable($foo, "foo");
 

  $foo = "string";
  $bar = ["key" => "value"];
  $baz = new stdClass();
  echo (new \Ice\Dump())->vars($foo, $bar, $baz);
Sleet usage: 

  {{ dump('str', 1, 2.5, true, null, ['key': 'value']) }}
